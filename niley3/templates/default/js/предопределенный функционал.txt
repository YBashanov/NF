подключение файлов, находящихся в папке js:
<script src='{BASE_JS}мой_файл.js' type='text/javascript'></script>


файлы: 
runonload.js - запускает функционал после загрузки страницы
runonload_L.js - вспомогательный файла для runonload.js, 
	хранит настройки некоторых параметров (необязателен к использованию)

	
функции:
a(переменная); - подсветка содержимого переменной, аналог a или v, используемых для подсветки данных в config
f(переменная); - форматированный вывод содержимого переменной
	f(переменная [, 'function'] [, 'search_string']); 


глобальные переменные:
get 	- содержит GET-массив
language- содержит метку языка (ru, en, de) обычно = ru
page	- содержит название страницы, открытой в данный момент (! при открытии главной = index)
