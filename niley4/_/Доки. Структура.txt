---   Описание структуры папок и назначения ---
2017-07-10

    - _ - папка, которая должна находиться лишь локально. Тут хранятся заметки, барахло

        - materials - файлы разных лет
        - scripts - вспомогательные скрипты, html-файлы

            - dumper.php - файл создания/распаковки дампа бд
            - migrations.php - файл наката миграций. Подключается к бд проекта

    - _old (2017...) - папка для старого проекта, который можно распаковать и посмотреть
    - _version - для поддержки версионности, чтобы последние файлы (функционал) был доступен с сервера в любое время

        - download_files.php - сюда будут записываться загруженные файлы (в конец файла)
        - id.php - текущая версия фреймворка (после загрузки обновляется)
        - index.php - скрипт для скачивания файлов с сервера
        - index_server.php - скрипт для отдачи файлов с сервера
        - index_upload.php - скрипт для загрузки файлов на сервер
        - index_upload_server.php - скрипт для получения файлов на сервере
        - version 4.0 - Niley.txt - описание в старом формате
        - version_client.php - конфигурационный файл клиента
        - version_server.php - конфигурационный файл сервера

    - config - настройки параметров сайта. Базовые пути до важных файлов, переменные, базы данных

        - db - настройки базы данных и стартовый пакет таблиц
        - management (в стиле niley3) - настройки подключаемого контента
        - site - конфигурационные файлы (основные)

    - php - бэкенд, набор функционала php
        варианты: в стиле niley3, niley4

        - ajax - здесь обрабатываются асинхронные запросы
        - data - предопределенные данные, без использования базы данных
        - libraries - набор классов, библиотек
        - modules - отдельные модули (пакеты на основе библиотек, объединяющие несколько библиотек), готовые к работе

    - view - фронтенд

        - css - набор общих для всех шаблонов стилей
        - error - страницы php для перенаправления на template-html файлы
        - fonts - подключаемые для всех шаблонов шрифты
        - img - общие для всех шаблонов картинки
        - js - общие библиотеки, классы
        - modules - готовые модули, использующие несколько js, css библиотек, готовые к работе
        - templates - различные шаблоны отображения страниц

            - niley3 - Шаблон. Пример "классического" фронтенда в стиле niley3
                Структура каждого шаблона внутри может быть разной.
                Описание прилагается к тому или иному стилю шаблона

                - css - стили шаблона
                - error - готовые страницы html для вывода сообщений об ошибке
                - html - html шаблона
                - img - картинки шаблона
                - js - файлы шаблона

        - uploads - загруженные файлы (изображения, звуки, видео)
